## Descripción
Se diseñó e implementó una base de datos relacional para simular el registro de usuarios y el proceso de compras de un sistema e-commerce.  
El proyecto incluye el modelo relacional, el script SQL con creación de tablas, inserción de datos y consultas solicitadas, además del documento PDF de identificación de entrega.
